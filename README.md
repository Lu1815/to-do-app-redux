# ToDo App Redux Test

![[Working App]](https://im5.ezgif.com/tmp/ezgif-5-7e94cc3d34.gif)

To run this app, first run the comman

```
 npm install
```

Or the equivalent depending on the package manager you use (yarn, pnpm).

To run the test, run the comman

```
 npm run test
```
